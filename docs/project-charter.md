# Project Charter - Sistema de Gestión de Inventario y Ventas

## Objetivo General
Desarrollar un aplicativo de software para la gestión de inventario y ventas de envases reutilizables para la empresa "El Serranito del Norte".

## Alcance
- Módulo de inventario (CRUD de envases)
- Módulo de ventas (registro de pedidos, actualización de stock)
- Módulo de clientes (registro y consulta)
- Módulo de reportes básicos

## Interesados (Stakeholders)
- Dueño de "El Serranito del Norte"
- Equipo de desarrollo (3 integrantes)
- Docente del curso

## Riesgos identificados
- Curva de aprendizaje de JavaFX
- Conexión a base de datos
- Disponibilidad del personal para pruebas

## Entregables principales
- Código fuente en GitHub
- Manual de usuario básico
- Presentación final
